## 기능 목록

### 입출력 및 초기 설정

- [ ] 입력 및 출력 뷰(View) 구현
- [ ] 입력 문자열 파서(Parser) 구현 (쉼표 분리, 공백 제거 등)

### 코치(Coach) 및 도메인

- [ ] 코치 이름 검증 및 일급 컬렉션(Coaches) 생성
    - 예외: 이름 길이(2~4), 최소 인원(2~5), 중복 불가
- [ ] 코치별 못 먹는 메뉴 검증 및 저장
    - 예외: 없는 메뉴, 개수(0~2)

### 추천 로직

- [ ] 요일별 카테고리 선정 기능 구현 (Randoms.pickNumberInRange)
    - 예외: 한 주에 같은 카테고리 최대 2회
- [ ] 코치별 메뉴 추천 로직 구현
    - 예외: 싫어하는 메뉴 제외, 해당 주 중복 메뉴 제외
- [ ] 추천 결과 저장(History) 및 조회 기능

### 메인 실행

- [ ] 컨트롤러(Controller) 구현 및 전체 흐름 연결
    - 예외 발생 시 에러 출력 후 재입력(Retry) 로직 적용

#### 참고 사항

- 코치의 이름은 2 ~ 4글자
- 코치는 최소 2명 ~ 5명
- 못먹는 음식은 최소 0 ~ 최대 2개
- 먹지 못하면 빈 값
- 중복되는 메뉴 추천하면 안 됨
- 한 주에 같은 카테고리는 최대 2회
- 잘못된 값인 경우 [ERROR] 메시지 출력 후 그 부분부터 다시 입력
